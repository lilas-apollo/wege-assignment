# WEGE Assignment

## Prerequisites

- Node.js >= 20.x (recommended: 20.5)
- npm >= 9.x
- Optional: VS Code or any IDE with TypeScript support

## Tech Stack

- Next.js 16 (App Router)
- TypeScript (strict mode)
- Tailwind CSS (for styling)
- TanStack Query (for data fetching, loading and caching)
- Lodash (for debounce)
- Lucide React (for icons)
- Zustand (for state management)
- React Select (for multi-select)

## Pages

- Home page: Display products with search box, filters and pagination.
- Product Details page: Display product details with Add to cart button.
- Cart page: Display added items with sub and total prices in addition to ability to increase and decrease item quantity or remove any product.

## Bonus Points

- Add a text search box with simple debouncing in Home page.
- Persist the cart to localStorage.
- Ensure good accessibility.

## Setup and Run

1. Clone the repository:
   ```bash
   git clone https://github.com/lilas-apollo/wege-assignment.git
   ```
2. npm install
3. npm run dev
